---+ NONCE -- generate a nonce (developers only)

*DEVELOPERS ONLY*

This macro is used to generate a 'nonce', a unique key that identifies the
source of a request in order to block some kinds of cyber attack.
See https://foswiki.org/Support/HowToIntegrateWithRequestValidation
for more information.
